# career-decision-simulator
ABSTRACT :
The Career Decision Simulator is a web application that helps users make smart career choices. It predicts future outcomes based on users' skills, interests, and proficiency levels. The system collects information like skill domain, skill level (basic, intermediate, advanced), and interest level (low, medium, high) to generate personalized career insights. 
The frontend uses React to offer users a simple and user-friendly interface. When a user submits their information, the system sends it as an HTTP request to the backend. The backend, built with Flask, processes this data and sends it to a simulation engine. The simulation engine is the core part of the system. It evaluates the input parameters and uses logic and data models to simulate different career paths. It takes into account factors such as salary growth, job demand, and automation risk based on the user's profile. The results are then organized into a JSON response and sent back to the frontend. 
On the frontend, the response data is presented with charts and interactive components. This allows users to understand trends and compare various career options. The system provides outputs such as predicted salary growth, job demand trends, automation risk, and comparisons among several careers. It also offers suggestions for skill improvement to help users achieve their desired career paths. The main aim of this project is to connect user potential with market needs by providing a decision-making tool based on simulations.
